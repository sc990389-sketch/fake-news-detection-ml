# 📰 Fake News Detection ML

A Machine Learning-powered web application that detects whether a news article is **Real** or **Fake** using Natural Language Processing (NLP) and a trained classification model.

## 🚀 Live Demo

Deploy the application on Streamlit Cloud and access it from anywhere.

## Features

* Fake News Classification
* Text Preprocessing
* TF-IDF Vectorization
* Machine Learning Prediction
* Interactive Streamlit Interface
* Real-Time Analysis

## How It Works

1. Enter a news article in the text box.
2. The text is cleaned and preprocessed.
3. TF-IDF converts text into numerical features.
4. The trained ML model predicts the result.
5. The app displays whether the news is Real or Fake.
